# Deployment Guide for Render

## Quick Summary: Local vs Production

### Local Testing (Currently)
```bash
# 1. Make sure backend is running
python manage.py runserver

# 2. Access via browser at:
http://localhost:8000
```

**Do NOT** open `index.html` directly. Always go through `http://localhost:8000` or the Django server will not process templates.

## Deploying to Render

### Prerequisites
1. Render account (free tier available at render.com)
2. GitHub repository with your code pushed
3. MySQL database (Aiven or another provider)

### Step-by-Step Deployment

#### Option A: Automatic Deployment (Recommended)
1. Go to [render.com](https://dashboard.render.com)
2. Click "New +" â†’ "Blueprint"
3. Select your GitHub repository
4. Render will use the `render.yaml` file to set up everything automatically

#### Option B: Manual Deployment
1. Create a new Web Service on Render
2. Connect your GitHub repository
3. Set the Build Command: `pip install -r requirements.txt && python manage.py collectstatic --noinput`
4. Set the Start Command: `gunicorn serrafins_site.wsgi:application --timeout 120`
5. Add Environment Variables (see below)

### Required Environment Variables on Render

```env
DEBUG=false
DJANGO_SECRET_KEY=<auto-generated-by-render>
DATABASE_URL=mysql+pymysql://user:password@host:port/dbname
ADMIN_PASSWORD=qwertyuiop
ADMIN_EMAIL=m.d.s.chamath@gmail.com
CNN_MODEL_URL=<your-google-drive-or-custom-url>
ALLOWED_HOSTS=yourdomain.onrender.com
```

### Important Notes for Render

1. **Database**: Use your existing Aiven MySQL connection via `DATABASE_URL`
   - Format: `mysql+pymysql://user:password@host:port/dbname`
   - Port for Aiven MySQL: usually 3306

2. **Static Files**: 
   - WhiteNoise middleware will serve them automatically
   - Don't worry about `collectstatic` errors for missing files

3. **Model File**: 
   - The `.pt` file will NOT be pushed to Render
   - Set `CNN_MODEL_URL` to download it at runtime or store it in `.gitignore`

4. **First Deploy**: Takes 5-10 minutes
   - Migrations run automatically via `release` command in Procfile

5. **Logs**: View via Render dashboard â†’ Logs tab

## Testing Locally Before Deploy

```bash
# 1. Install production packages
pip install gunicorn whitenoise

# 2. Collect static files
python manage.py collectstatic --noinput

# 3. Test with production WSGI server
gunicorn serrafins_site.wsgi:application --bind 0.0.0.0:8000

# 4. Visit http://localhost:8000
```

## Common Issues

| Issue | Solution |
|-------|----------|
| Static files not loading | Run `collectstatic`, ensure `STATIC_ROOT` is set |
| Database connection fails | Verify `DATABASE_URL` format and network access |
| "DisallowedHost" error | Add domain to `ALLOWED_HOSTS` in environment |
| Model file 404 | Check `CNN_MODEL_URL` is accessible and not behind auth |
